

# COVID19 report Android Kotlin MVVM Project

# 簡介
此專案使用 Kotlin 練習 MVVM Databinding 架構專案，也因應新冠漢炎的疫情，實作了以下功能：
* 全球新冠肺炎資訊
<br>搜尋國家功能僅支援英文
* 新冠肺炎相關新聞
* 簡易版台灣口罩地圖
<br>僅支援 Locale.country 在 tw，其他地區不支援
* 多國語言僅支援中英文
